# Dental Management System Case Study

## The Project

The Professional Dental Appointment Management System is a comprehensive web application designed to streamline dental practice operations. This full-stack solution provides a complete business platform for dental practices, featuring patient management, appointment booking, secure authentication, and administrative tools.

## Technical Implementation

### Full-Stack Architecture

Built using modern web technologies for scalability and performance:

```typescript
// Authentication middleware with JWT tokens
export const authenticateToken = (req: Request, res: Response, next: NextFunction) => {
  const authHeader = req.headers['authorization'];
  const token = authHeader && authHeader.split(' ')[1];

  if (!token) {
    return res.status(401).json({ message: 'Access token required' });
  }

  jwt.verify(token, process.env.JWT_SECRET!, (err: any, user: any) => {
    if (err) return res.status(403).json({ message: 'Invalid token' });
    req.user = user;
    next();
  });
};
```

### Database Architecture

PostgreSQL with TypeORM for robust data management:

```typescript
// User entity with role-based permissions
@Entity('users')
export class User {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column()
  name: string;

  @Column({ unique: true })
  email: string;

  @Column()
  password: string; // Hashed with bcrypt

  @Column({ default: 'patient' })
  role: 'patient' | 'admin' | 'doctor';

  @Column({ nullable: true })
  phone: string;

  @OneToMany(() => Appointment, appointment => appointment.patient)
  appointments: Appointment[];
}
```

- Input validation with Joi schemas
- SQL injection protection via TypeORM

- Database connection encryption
